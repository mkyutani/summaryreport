# Output Format

Use this format unless the user asks for a different structure.

## 1. ページの概要

- このページまたは文書が何についてのものかを簡潔に説明する。
- 主催者・会議体・対象テーマなど、文脈把握に必要な情報を含める。

## 2. 要約（Abstract）

- コードフェンス内に記載する。
- 目安は約1,000文字（日本語）。
- 結論・主要論点・重要事項を短くまとめる。

## 3. ページの詳細サマリー

- トピックや議題ごとに整理して記載する。
- 主要な根拠や補足資料の内容を必要に応じて含める。
- 事実ベースで簡潔に書く。
